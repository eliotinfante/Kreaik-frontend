Política de Privacidad

Última actualización: Junio 2025

1. Información que Recopilamos

Recopilamos la siguiente información cuando usas Kreaix:

  • Datos de cuenta: nombre, email y contraseña (encriptada)
  • Datos de uso: prompts enviados (primeros 120 caracteres), modelos usados, créditos consumidos
  • Datos de pago: historial de recargas (sin datos de tarjeta — procesados por Lemon Squeezy)
  • Datos técnicos: dirección IP, tipo de navegador, sistema operativo

2. Cómo Usamos tu Información

  • Para operar y mejorar el servicio
  • Para procesar pagos y gestionar créditos
  • Para enviarte emails transaccionales (confirmaciones, alertas de créditos bajos)
  • Para detectar y prevenir fraudes
  • Para cumplir obligaciones legales

3. Compartición de Datos

Compartimos datos limitados con proveedores de servicios necesarios para operar Kreaix:

  • Anthropic, OpenAI, Google: reciben tus prompts para generar respuestas
  • Lemon Squeezy: procesa pagos (tienen su propia política de privacidad)
  • Resend: envía emails transaccionales
  • Railway: infraestructura del servidor

No vendemos ni compartimos tus datos personales con terceros para fines publicitarios.

4. Seguridad de los Datos

Protegemos tus datos mediante:

  • Contraseñas encriptadas con bcrypt
  • Comunicaciones cifradas con SSL/TLS
  • Tokens de sesión seguros (JWT)
  • Acceso restringido a la base de datos

5. Tus Derechos

Tienes derecho a:

  • Acceder a tus datos personales
  • Corregir información incorrecta
  • Solicitar la eliminación de tu cuenta y datos
  • Exportar tus datos

Para ejercer estos derechos, contacta a: privacidad@kreaix.com

6. Cookies

Usamos localStorage del navegador para mantener tu sesión activa y recordar tus preferencias de tema. No usamos cookies de seguimiento publicitario.

7. Retención de Datos

Mantenemos tus datos mientras tu cuenta esté activa. Si eliminas tu cuenta, borramos tus datos personales en un plazo de 30 días, conservando solo los registros contables requeridos por ley.

8. Contacto

Para preguntas sobre privacidad: privacidad@kreaix.com
